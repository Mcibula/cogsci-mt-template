\setglossarypreamble[symbols]{
    \noindent
    Capital letters (e.g., $X$) are used for random variables, sets, or matrices if boldfaced (e.g., $\bm{M}$).
    Lower-case letters denote scalars (e.g., $x$), and the values of random variables (e.g., $X = x$).
    Vectors are denoted by lower-case bold letters (e.g., $\bm{v}$).\par
}
